# Socket

Here,the client (SocketText.java) sends a number (message) to the server, Server receives this number and multiplies it by 2. Server (Server.java) sends back the result (message) to the client (SocketText.java).

Validation: In case the number sent by the client was not a proper number, server (Server.java) sends back the message “Please send a proper number” to the client (Client.java)

Input : A Integer Number.
Output : A number multipied by 2 send by server
